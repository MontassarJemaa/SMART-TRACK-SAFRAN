# SAFRAN SMART TRACK Web Dashboard

Dashboard web Next.js pour la gestion des outillages RFID.

## Stack
- Next.js 14 App Router
- TypeScript strict
- Tailwind CSS
- Redux Toolkit
- React Query
- Supabase
- NextAuth
- Recharts
- jsPDF + xlsx

## Setup

1. Copier `.env.example` en `.env.local`
2. Remplir les variables d'environnement Supabase et NextAuth
3. Installer les dépendances:
   ```bash
   npm install
   ```
4. Lancer le projet:
   ```bash
   npm run dev
   ```
